https://my.paystar.ir/

== Description ==

with this plugin, you can easily add payment form (like donate) with paystar.ir online gateway
با این پلاگین میتونید فرم های برای دریافت مبالغی از کاربران سایت (مثل فرم های اهدای وجه) با درگاه آنلاین پی استار راه اندازی کنید


== Installation ==
 
use shortcode [paystar] to show payment form in your post/pages or in themes. dont forget to enter paystar.ir terminal (API-Key) in plugin settings
شرتکد [paystar] را برای نمایش فرم پرداخت در برگه / پست مورد نظر وارد کنید
فراموش نکنید که کد ترمینال پذیرندگی خودتون رو در تنظیمات افزونه وارد کنید
تنظیمات افزونه در یک منوی جدید که به منوهای ادمین وردپرس اضافه خواهد شد ، قابل دسترسی ست
